# LCD 3.5" TFT_eSPI ESP32 - ARDUINO IDE

### TFT LCD 2.8", 3.2", dan 3.5"

## Wiring Diagram | LCD 3.5 Touch Screen ILI9488 480x320

- 5V		-->	- VCC
- GND		-->	- GND
- 32		-->	- CS
- 33		-->	- RESET
- 25		-->	- DC/RS
- 26		-->	- SDI(MOSI)
- 27		-->	- SCK
- 5V		--> 	- LED
- NO CONNECTION	-->	- SDO (MISO)
- 27		-->	- T_CLK
- 14		-->	- T_CS
- 26		-->	- T_DIN
- 12		--> 	- T_DO
- NO CONNECTION --> 	- T_IRQ
